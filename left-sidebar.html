Digital Forensics

Digital Evidence Handling & Crime Scene Processing

Project Summary

This project focuses on a digital forensic investigation of a suspected insider data breach in a corporate environment. The case involved a powered-on workstation, a connected SanDisk 32GB USB device, deleted client database evidence, suspicious email communication, and possible cloud-based data exfiltration through Dropbox and Google Drive.

View Digital Forensics PDF

What I Did

Identified Evidence

I identified important digital evidence such as system logs, USB logs, browser history, suspicious email, deleted files, RAM, and network traffic.

Followed Volatility Order

I prioritized volatile evidence such as live network traffic and RAM before collecting non-volatile storage evidence.

Recovered Deleted Evidence

I analyzed forensic artifacts and recovered deleted data such as client_database_deleted.csv using Autopsy.

Correlated Findings

I connected system login activity, USB insertion, suspicious email, and browser history to build a clear timeline of the incident.
